Average4K

Average4K is a Multiplayer Focused Competitive rhythm game built on its own engine from the ground up in cpp. Allowing for Steam Workshop Support for Chart Packs, and Skins.

Average4K also allows for full customization of the game with it's skins and menus, and even lets players script their own modcharts.

Vs. Tricky

Vs. Tricky is a Friday Night Funkin' mod built on Haxe using the Flixel Rendering Framework using the original source code as a base.

This is also built on an "engine" of mine called Kade Engine, which itself is a mod of Friday Night Funkin' as well.

The mod is about a character from the internet series Madness called Tricky.

I was the main programmer on this

Vs. Hex

Vs. Hex is a Friday Night Funkin' mod also like Vs. Tricky. Using the same framework and "engine." as Vs. Tricky. This did come out after Vs. Tricky v2, but was a standalone mod before it as well.

The mod is about a robot who likes to play basket ball, and it even has a super cool modchart made by TaroNuke (the creator of a rhythm game called NotITG) which is pretty cool.

Very proud of this!

I was the main programmer on this
